Roller Ring Hardware
=============================================

### Description

- Components, outside of the belt and CASM, were toleranced and printed around using PLA ([Bambu PLA Basic](https://us.store.bambulab.com/products/pla-basic-filament?srsltid=AfmBOorc7Z0PmoGqhGUEHbTq-vpCCKCEAU0vk6l4jsVQA2-2yhMzEfRV)) on a Bambu X1C. The belt and CASM were toleranced and printed around using TPU ([NinjaTek Chinchilla 75A](https://ninjatek.com/shop/chinchilla/)) on a Bambu A1.
- Other filaments can be utilized as long as the material properties are comparable to the aforementioned filaments 
- In the event of using different printers, some light sanding may be required to ensure component can be installed.
