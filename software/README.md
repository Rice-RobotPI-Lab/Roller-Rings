Roller Ring Software
=============================================

### Description


- Low level control, and python API used for the project are provided in [rgSerial](https://github.com/yuanshenli/rgSerial/tree/rgserial_4motor). This is the same codebase used in both the [Tactile-Reactive Roller Grasper](https://yuanshenli.com/tactile_reactive_roller_grasper.html) and the [Belt–Augmented Compliant Hand](https://yuanshenli.com/bach.html). 
- The gerber file for the PCB can be found in [Tactile-Reactive-Roller-Grasper repository](https://github.com/yuanshenli/Tactile-Reactive-Roller-Grasper) <br>
- For controller and keyboard use, you'll need to install pygame>=2.1.0 to interface with the motors

For further questions and inquiries regarding the hardware detailed here, please contact [Shenli Yuan](https://yuanshenli.com/).
